# CppRockPaperScissors

CppRockPaperScissors is a simple console-based Rock-Paper-Scissors game written in C++.
It allows a player to compete against the computer in a single round of the classic game.

*****************************************************************************************************

# Features

- Console interface for simple interaction

- Random computer moves

- Displays player and computer choices clearly

- Announces the winner after each round

- Easy to read and well-commented code for beginners
- 
*****************************************************************************************************

# How to Run

Clone or download the repository

Compile the program using a C++ compiler:

g++ CppRockPaperScissors.cpp -o CppRockPaperScissors


Run the program:

./CppRockPaperScissors


Follow the prompts to choose rock, paper, or scissors
